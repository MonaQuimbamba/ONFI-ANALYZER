ONFIAnalyzerSettings::ONFIAnalyzerSettings ( )

consctructor

it'll build the settings for our analyzer , set up each signal on one channel

Parameters
mALEChannelAddress latch enable: Loads an address from I/O[7:0] into the address register
mCEChannelChip enable: Enables or disables one or more die (LUNs) in a target.
mCLEChannelCommand latch enable: Loads a command from I/O[7:0] into the command register.
mLOCKChannelWhen LOCK is HIGH during power-up, the BLOCK LOCK function is enabled. To disable the BLOCK LOCK, connect LOCK to VSS during power-up, or leave it disconnected (internal pull-down).
mREChannelRead enable: Transfers serial data from the NAND Flash to the host system.
mWEChannelWrite enable: Transfers commands, addresses, and serial data from the host system to the NAND Flash.
mWPChannelWrite protect: Enables or disables array PROGRAM and ERASE operations
mIOChannelData inputs/outputs: The bidirectional I/Os transfer address, data, and command infor- mation.
mRBChannelReady/busy: An open-drain, active-low output that requires an external pull-up resistor. This signal indicates target array activity.
